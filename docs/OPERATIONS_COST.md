# 运营成本分析文档 (Operations Cost Analysis)

## 1. 固定成本 (Fixed Costs)

| 项目 | 费用 (预估) | 周期 | 说明 |
| :--- | :--- | :--- | :--- |
| **云服务器 (VPS)** | ¥30 - ¥100 | 每月 | 用于部署定时任务 (Cron Job) 和 Python 脚本。推荐腾讯云轻量应用服务器或 Vercel (免费版可用)。 |
| **域名 (可选)** | ¥50 | 每年 | 如果需要搭建 Web 展示页。 |

## 2. 变动成本 (Variable Costs)

### 2.1 DeepSeek API (LLM)
*   **定价**: 输入 ¥1.0/M tokens, 输出 ¥2.0/M tokens。
*   **消耗估算**:
    *   假设每天处理 20 条新闻，每条原文 2k tokens，摘要生成 500 tokens。
    *   日消耗: $20 \times (2000 \times 1 + 500 \times 2) / 1000000 \approx ¥0.06$。
    *   **月消耗**: $¥0.06 \times 30 \approx ¥1.8$。
*   **结论**: 极其低廉，个人开发者完全负担得起。

### 2.2 Jina AI Reader API
*   **定价**: 基础版免费，高级版按请求次数收费。
*   **策略**: 优先使用免费额度，若超量可考虑自行部署爬虫 (如 Firecrawl)。

## 3. 隐性成本 (Hidden Costs)

*   **开发维护**: 脚本报错、网页结构变更导致爬虫失效的修复时间。
*   **内容审核**: AI 生成的内容可能存在幻觉或敏感信息，发布前最好人工过目 (Human-in-the-loop)。

## 4. 盈利模式规划 (Revenue Model)

1.  **流量主**: 公众号/头条号广告分成。
2.  **付费订阅**: 提供更深度的行业分析报告 (Pro版)。
3.  **分发渠道**: 将整理好的数据通过 API 卖给其他聚合平台。
